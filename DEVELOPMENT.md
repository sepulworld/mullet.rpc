### Local Development

#### Prereq

##### Install Docker for Mac

```
open https://docs.docker.com/desktop/mac/install/
```

##### Install Tilt 
```
curl -fsSL https://raw.githubusercontent.com/tilt-dev/tilt/master/scripts/install.sh | bash
```

##### Tilt Up 

```
tilt up
```

#### Register a user

http://localhost:8080/auth/login?next=%2Findex
